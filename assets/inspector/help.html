Image Processor

In this application you can manipulate images using filters and trasform tools. By default you see the example where you can create token for various TTRPG from several images.

Help

Upload - you can upload an image via Upload node using context menu or url. Alternatively, you can drag and drop images from your computer directly into the application.
Download - you can download images from Display nodes using context menu.
Files - you can save and load a file with your workflow using dropdown in the toolbar. Produced files have ".imp" extension.
Nodes - you can get an additional info about every node by clicking on the question mark symbol of the node.
